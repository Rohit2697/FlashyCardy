# shadcn/ui Rule

Use these commands when working with shadcn/ui in this project.

## Initialize

```bash
npx shadcn@latest init
```

## Add Components

```bash
npx shadcn@latest add [component]
```

## Example

```bash
npx shadcn@latest add card
```

## Notes

- Use the exact commands above when installing or adding shadcn/ui pieces.
- Add any future shadcn-specific conventions to this file so it remains the single source of truth.
